# 🧠 AI-Powered Skin Cancer Detection System

An Artificial Intelligence based medical image analysis system that detects different types of skin cancer using **Deep Learning (Convolutional Neural Networks)**.

This project demonstrates how AI can assist healthcare professionals by analyzing dermoscopic images and providing predictions for different skin cancer classes.

---

# 🔬 Project Overview

Skin cancer is one of the most common forms of cancer worldwide. Early detection plays a critical role in successful treatment.

This project uses **CNN-based deep learning models** to classify skin lesion images into multiple categories, helping demonstrate the potential of AI in medical imaging and healthcare diagnostics.

---

# ⚙️ Technologies Used

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

# 🧠 Deep Learning Model

The model is built using **Convolutional Neural Networks (CNN)** which are highly effective for image classification tasks.

Model Capabilities:

- Skin lesion image classification
- Multi-class prediction
- Confidence score output
- Performance evaluation using confusion matrix

---

# 🖥️ Web Application

The project includes a **Streamlit-based interactive web application** where users can:

- Upload a skin lesion image
- Run AI analysis
- View predicted disease class
- See model confidence score
- View performance metrics such as accuracy and confusion matrix

---

# 📊 Model Performance

Evaluation Metrics:

- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1 Score

These metrics help measure the reliability of the AI model.
